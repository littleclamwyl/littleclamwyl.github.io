二、 注释

ASCII最大字符数256 -> UNICODE 65536个字符，可以包含所有语言的所有字符，就不会乱码了。

UNICODE只规定了每个字符的数字编码，但没有规定这个字符如何存储。

  1. utf-8/utf-16/utf-32是Unicode的一种具体实现

  2. gbk是中文编码
